# vagrant-centos7-wl-jrx
Centos7 with:
 - docker
   - docker-compose
 - git
 - nodejs
   - npm
   - yarn
 - vscode (https://code.visualstudio.com/)
 - X11
 - Cinnamon desktop
 - zsh
   - custom alias (./files/.zshrc)
   - ho-my-zsh (https://ohmyz.sh/)

## Prerequisites
Vagrant plugins:
 - vagrant-vbguest
 - vagrant-proxyconf (optional)

## Install
Just "vagrant up" then "vagrant reload" and enjoy !
